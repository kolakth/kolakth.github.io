Smart electric saving at home

Smart electric saving at home is a senior project and also hatch startup program at KMUTT. This project objective is to convince people to reduce the energy usage at their house. In order to convince user is to show their daily usage and prediction model within app itself.

The app call “EnergyWatch” which only operated on iOS and also a sensors to capture energy consumption from main breaker unit. I responsible for creating a wireframing and features to high fidelity later with rapid prototyping method by hatch program.

The App is developed using Swift 4.0 and Google Firebase as a back-end and also realtime database. The sensors itself can connect to firebase and upload data every seconds. So users can see their usage in real-time on the app instantly.

 Real-time daily consumption

This is the homepage after user logged in. They can see their daily useage right away with real-time graph. User can also check for prediction model and previously usage.

 See previous usage

In the Energy Saver tab, user can select the data on the calendar to check electricity usage for each day in months. It’s show units, and how much it cost you.

 See estimated billing

In the Energy Saver tab, user can select the data on the calendar to check electricity estimated billing for each day in months. And also by the estimated bill for end of the month if you keep using with this behavior.
